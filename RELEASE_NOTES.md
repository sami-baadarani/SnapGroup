### ✨ What's New

- **Automatic updates** — SnapGroup now keeps itself up to date using [Sparkle](https://sparkle-project.org). When a new version is available you'll be notified and can install it in a click — no manual reinstall needed.
- **"Check for Updates…"** — a new item in the menu-bar menu lets you check on demand.

From this release onward, updates are delivered automatically — notarized and cryptographically verified with an EdDSA signature.
